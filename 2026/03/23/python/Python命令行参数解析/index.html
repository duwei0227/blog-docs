Python命令行参数解析

Python的argparse模块是标准库中用于解析命令行参数的核心工具。它能够自动生成帮助信息、处理无效参数，并支持位置参数、可选参数、默认值等功能。

ArgumentParser是argparse模块的核心类，用于创建命令行解析器。
