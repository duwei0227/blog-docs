Python CSV文件读写

CSV（Comma Separated Values）格式是电子表格和数据库中最常见的输入输出文件格式。Python的csv模块提供了读取和写入CSV格式表格数据的功能，让程序员可以专注于数据处理而无需关心CSV格式的细节。
