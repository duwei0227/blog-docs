Python urllib HTTP请求指南

urllib 是 Python 标准库中用于处理 URL 的模块，包含 urllib.request（发送请求）、urllib.parse（URL 解析和编码）、urllib.error（异常处理）三个子模块。无需安装第三方库即可发送 HTTP 请求。
